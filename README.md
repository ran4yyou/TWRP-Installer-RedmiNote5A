## OrangeFox Installer for Xiaomi Redmi Note 5A (UGGLITE)
Simple Windows batch script to automatically flash and boot OrangeFox Recovery on Redmi Note 5A.

## Features
- Auto detect device via ADB
- Auto reboot to fastboot mode
- Flash and boot OrangeFox recovery
- Clean and easy to use (just one click)
- Made by **Wildan Hermawan**

## Requirements
- Unlocked bootloader
- ADB & Fastboot drivers installed
- USB Debugging enabled on your phone
